# material3 — Agent Instructions

## Material 3 Component Lookup

When adding or aligning components, search in <https://m3.material.io/> to find the component first. On each component's overview page, you can usually find links or references to both its Jetpack Compose implementation (almost identical to Compose Multiplatform's Compose UI APIs) and its web/Material Web implementation.

For an up-to-date list of supported unified component APIs, refer to README.md in the repository root.
